# proyecto EmailsP
EmailsP es una API diseñada para el envío masivo de correos electrónicos con fines académicos y no lucrativos. Construida siguiendo los principios de Domain-Driven Design (DDD), esta solución está pensada para facilitar la actualización constante del software, garantizando flexibilidad, escalabilidad y mantenimiento eficiente.

# Características principales
- Envío masivo de correos: Permite enviar correos electrónicos a múltiples destinatarios de forma eficiente.
- Diseño modular: Basado en DDD para una arquitectura limpia y mantenible.
- Fines académicos: Ideal para proyectos educativos, pruebas y entornos de aprendizaje.
- No lucrativo: Enfocado en usos sin ánimo de lucro, promoviendo el acceso a herramientas tecnológicas.

## Instalación
1) Clona el repositorio:
git clone https://github.com/SebastianGallegoC/proyectoEmails.git

2) Navega al directorio del proyecto:

cd EmailsP

Ejecuta la solucion del proyecto en Visual Studio Code Community:
Se recomienda instalar y tener actualizado las dependencias 
- "Desarrollo ASP.NET y web".
- "Desarollo de Azure".
- "Desarrollo de Escritorio de .NET".
